常用命令

  1. 初始化，添加修改，提交修改，推到服务器

  2. 拉取线上特定分支到本地

  3. 推送本地分支到线上

  4. 换行符转换的问题

    • 设置自动转换选项autocrlf
    • $ git config –local core.autocrlf true | input | false
      1. true 表示开启自动转换，迁入时将文件换行风格转换成Unix风格，迁出时根据本地系统确定是否转换成CRLF
      2. input 表示迁入的时候将换行风格转换成Unix风格，迁出时不做处理。
      3. false 表示迁入迁出都不对换行风格进行处理

  5. 清空git缓存

  6. 重新再次输入密码
